白っぽい文字で動作するプログラミング言語です。Brainf*ckの各命令に使用する文字を置き換えた派生言語です。各命令の対応は以下の通りです。本体はドットインストール Brainf*ck入門とほとんと同じです。「サンプルをロードする」ボタンを押すと、Hello,world!を出力するプログラムをロードします。

  1. 白 : > : increment
  2. 臼 : < : decrement
  3. 日 : + : forward
  4. 曰 : - : back
  5. 目 : , : input
  6. 且 : . : output
  7. 旦 : [ : start loop
  8. 亘 : ] : end loop
